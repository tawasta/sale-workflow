Sale Order Partner Tag to CRM Tag
=================================
* Tag from partner category to sale order CRM tag 

Configuration
=============
1. Enable "Tag to Sale Order" for a specific Partner Category.
2. Create a new sale order for partner to automatically fill CRM Tag from Partner Category.
