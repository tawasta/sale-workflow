===================================
Sale Order Delivery Date to Invoice
===================================

Move effective date, commitment date or expected date to invoice.

Prefer effective date, if it's set (a delivery is done).
If it is not set, use commitment date or expected date by selecting the
later one that is set (estimated delivery).
