====================================
Sale order: create pickings manually
====================================

Allows manually (re)creating pickings from SO.

Works in these situations:
- Delivery was cancelled and need to be recreated
- Delivery was deleted and needs to be recreated
- Delivery lines were deleted and need to be recreated
- A partial delivery was made but backorder wasn't created
