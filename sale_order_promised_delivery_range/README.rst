========================================
Sale Order: Promised Delivery Date Range
========================================

* Adds new fields for storing date range of promised delivery
* The promised delivery date is shown on sale prints

Configuration
=============
* None needed

Usage
=====
* Justreate a Sale Order and fill in the new fields
